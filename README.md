<h1 class="text-align">Nezuko API</h1>
<p class="text-align">Simple API Modules Npmjs</p>

Installation
```js
npm i nezuko-api
```
Usage
```js
const nezuko = require("nezuko-api")

nezuko.yta(YOUR_URL).then(async(res) => {
  // YouTube Download Audio
  console.log(res)
})
```
